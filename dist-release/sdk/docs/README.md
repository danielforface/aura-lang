# AuraSDK (Prototype)

This folder is bundled into the AuraSDK zip.

## First steps

- Run `aura init` to scaffold a project.
- Run `aura run main.aura --mode avm` for interactive prototyping.

## Environment

- `AURA_HOME` points at the AuraSDK root.
- `AURA_HOME/bin` should be on your `PATH`.
