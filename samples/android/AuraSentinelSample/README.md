# Aura Sentinel Sample (Android)

A minimal Android app project you can open in Android Studio and build into an APK.

## Build
- Recommended: Open this folder in Android Studio and run **Build > Build Bundle(s) / APK(s) > Build APK(s)**.

## CLI build (Windows)
If you installed the toolchain via `sdk/android/setup-android.ps1`, you can run:

`powershell -ExecutionPolicy Bypass -File sdk/android/build-apk.ps1`

## Aura
This sample includes a tiny Aura file in `app/src/main/assets/hello.aura` to demonstrate packaging Aura sources with an APK.
