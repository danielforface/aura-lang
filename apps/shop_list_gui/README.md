# Shop List (Windows GUI)

A simple Windows GUI tool to manage a shopping list with **real persistence**.

## Run

From the repo root:

```powershell
C:/Users/danie/Documents/code/lang/.venv/Scripts/python.exe apps/shop_list_gui/main.py
```

## Data location

By default it saves to:

- `%APPDATA%\AuraShopList\shopping_list.json`

Use **File → Open…** to load another JSON file, or **Import/Export** to move data.
